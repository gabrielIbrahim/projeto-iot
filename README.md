# 🌌 Dashboard IoT - Métricas Climáticas

Sistema de monitoramento climático em tempo real utilizando conceitos de IoT, Machine Learning e visualização de dados.

## 🚀 Funcionalidades

- 📡 Coleta de dados (simulada)
- 🌡️ Monitoramento de temperatura e umidade
- 📈 Dashboard em tempo real
- 🔮 Previsão com Regressão Linear
- 🚨 Detecção de anomalias (Isolation Forest)

## 🧠 Tecnologias

- Python (Flask)
- Pandas / NumPy
- Scikit-learn
- Chart.js
- JustGage

## ▶️ Como executar

```bash
pip install -r requirements.txt
python app.py
python simulator.py
